
## ⚡ 第一部分：数据规模推算法（看 $n$ 定算法）

如果你不知道该用 $O(n)$ 还是 $O(n^2)$，看一眼题目给的数据范围：

|**数据规模 n**|**允许的时间复杂度**|**可能的算法建议**|
|---|---|---|
|**$n \le 20$**|$O(2^n)$ 或 $O(n!)$|回溯法、排列组合、搜索|
|**$n \le 500$**|$O(n^3)$|动态规划、三重循环、Floyd算法|
|**$n \le 5000$**|$O(n^2)$|基础 DP、双重循环、简单模拟|
|**$n \le 10^5$**|$O(n \log n)$|**分治、排序、堆、二分查找、线段树**|
|**$n \le 10^6$**|$O(n)$|**双指针、滑动窗口、单调栈、哈希表、DFS/BFS**|

## 🎯 第二部分：关键词“暗号”匹配表

当你读题时，看到这些词，大脑要立即产生以下反射：

### 1. 数组 / 字符串 (Linear Data)

- **关键词：** “连续子数组”、“最长子串” $\rightarrow$ **滑动窗口**
    
- **关键词：** “有序数组”、“查找特定值” $\rightarrow$ **二分查找**
    
- **关键词：** “反转链表”、“寻找中点” $\rightarrow$ **快慢指针**
    
- **关键词：** “两数之和”、“重复元素”、“出现频率” $\rightarrow$ **哈希表**
    

### 2. 树 / 图 (Non-linear Data)

- **关键词：** “层序遍历”、“最短路径”、“最少步数” $\rightarrow$ **BFS (队列)**
    
- **关键词：** “所有路径”、“目标路径是否存在” $\rightarrow$ **DFS (递归)**
    

### 3. 最值问题 (Optimization)

- **关键词：** “最大/最小”、“第 K 个”、“前 K 个” $\rightarrow$ **堆 (Priority Queue)**
    
- **关键词：** “最大子序列和”、“爬楼梯方案” $\rightarrow$ **动态规划 (DP)**
    
- **关键词：** “局部最优得出全局最优” $\rightarrow$ **贪心算法**
    

### 4. 元素关系 (Relationships)

- **关键词：** “左/右边第一个更大/更小” $\rightarrow$ **单调栈**
    
- **关键词：** “找出所有可能的组合/排列” $\rightarrow$ **回溯法**
