	# Datacom Network Engineer 学习路线图（详细要求）

## 1. **网络认证证书**

### HCIA/CCNA 级别（入门必备）

**学习程度**：必须精通

- 掌握OSI七层模型和TCP/IP四层模型
- 熟练配置交换机VLAN、trunk、STP
- 掌握静态路由、默认路由配置
- 理解NAT、ACL基本原理
- 能独立搭建小型企业网络
- **目标**：通过HCIA或CCNA认证考试

### HCIP/CCNP 级别（核心技能）

**学习程度**：深入理解并能实际应用

- 精通OSPF：区域划分、LSA类型、路由汇总、虚链路
- 精通BGP：IBGP/EBGP、路由策略、AS-Path、Community属性
- 掌握MPLS VPN、GRE隧道技术
- 高级交换技术：VRRP/HSRP、堆叠、集群
- QoS流量管理和优化
- **目标**：能够设计和实施中大型企业网络

### HCIE/CCIE 级别（专家方向，可选）

**学习程度**：如有时间建议深造

- 复杂网络场景troubleshooting
- 多协议互通和路由优化
- 大规模网络架构设计
- **目标**：成为网络领域专家

---

## 2. **路由协议**

### BGP（边界网关协议）

**学习程度**：必须精通，工作核心技能

- **基础**：BGP邻居建立、状态机、消息类型
- **进阶**：路由选路原则（13条选路规则必须背下来）
- **高级**：
    - 路由策略：route-map, prefix-list, as-path-filter
    - BGP属性操控：Local-Preference, MED, AS-Path prepend
    - BGP反射器、联盟配置
    - BGP在数据中心的应用（Spine-Leaf架构）
- **实战**：能够快速定位BGP路由黑洞、环路问题
- **学习时间**：2-3个月深入学习

### OSPF（开放最短路径优先）

**学习程度**：必须精通

- **基础**：OSPF区域概念、LSA类型（1-7类）
- **进阶**：
    - 区域类型：骨干区域、标准区域、Stub、Totally Stub、NSSA
    - SPF算法原理
    - 路由汇总技巧
- **高级**：OSPF优化：邻居认证、网络类型调整、LSA过滤
- **实战**：能够诊断OSPF邻居无法建立、路由不通问题
- **学习时间**：1-2个月

---

## 3. **网络安全**

### 防火墙技术

**学习程度**：熟练配置和故障排查

- **基础**：
    - 防火墙工作原理（状态检测、包过滤）
    - 华为防火墙（USG系列）或思科ASA基本配置
    - 安全区域（Zone）概念
- **进阶**：
    - ACL规则编写（精确匹配业务流量）
    - NAT配置：源NAT、目的NAT、双向NAT
    - VPN配置：IPSec VPN、SSL VPN
- **高级**：
    - 入侵检测IPS规则调优
    - 应用层防护（DDoS防护、应用识别）
    - 防火墙双机热备（VRRP+状态同步）
- **实战要求**：能够快速定位防火墙阻断业务流量的原因
- **学习时间**：2个月

### TCP/IP 协议栈

**学习程度**：理论必须精通

- **必须掌握**：
    - TCP三次握手、四次挥手（抓包分析）
    - TCP滑动窗口、拥塞控制算法
    - IP分片与重组
    - ARP原理与ARP欺骗防护
    - ICMP协议（Ping、Traceroute原理）
- **深入理解**：
    - TCP重传机制（超时重传、快速重传）
    - TCP KeepAlive机制
    - MTU/MSS调优
    - 端口号范围（well-known、registered、dynamic）
- **实战**：使用Wireshark抓包分析网络问题
- **学习时间**：持续学习，1个月入门

## 4. **Linux 系统**

### Linux 基础

**学习程度**：熟练使用命令行

- **文件操作**：ls, cd, cp, mv, rm, chmod, chown, find, grep
- **文本处理**：cat, less, head, tail, awk, sed, vim
- **进程管理**：ps, top, kill, systemctl, crontab
- **网络命令**：
    - ping, traceroute, telnet, netstat, ss
    - tcpdump（抓包）
    - ip addr, ip route（查看和配置网络）
    - curl, wget（测试HTTP连接）
- **系统监控**：df, du, free, iostat, vmstat
- **学习时间**：1个月基础 + 持续使用

### Linux 进阶

**学习程度**：能够编写简单脚本

- Shell脚本编写（Bash）：自动化日常运维任务
- SSH远程管理、密钥认证
- 日志分析：/var/log目录下的日志文件
- 防火墙：iptables基本规则
- **学习时间**：2-3个月

## 5. **Python 编程**

### Python 基础

**学习程度**：能够独立开发自动化脚本

- **基础语法**（1个月）：
    - 变量、数据类型、运算符
    - 列表、字典、元组、集合
    - if/for/while循环
    - 函数定义与调用
    - 文件读写操作

### Python 网络自动化

**学习程度**：这是工作核心技能

- **网络库**（2-3个月）：
    - **Paramiko**：SSH连接网络设备，执行命令
    - **Netmiko**：专为网络设备设计的SSH库
    - **Requests**：调用REST API（华为云API）
    - **NAPALM**：跨厂商网络设备配置管理
- **实战项目**：
    - 批量备份网络设备配置
    - 自动巡检脚本（检查接口状态、CPU、内存）
    - 告警处理自动化（解析告警，执行预定义操作）
    - 设备配置变更脚本（批量修改VLAN、路由）

### Python 数据处理

**学习程度**：用于报表和数据分析

- **Pandas**：处理Excel数据（工单统计、流量分析）
- **openpyxl**：操作Excel文件
- **Matplotlib**：生成图表（网络流量趋势图）
- **学习时间**：1-2个月

### 总体Python学习时间：4-6个月达到工作要求

## 7. **华为云与数据中心网络**

### 华为云基础

**学习程度**：了解架构和常用服务

- VPC（虚拟私有云）概念
- ECS（云服务器）、ELB（负载均衡）
- 华为云网络产品：云连接CC、云专线DC
- **学习时间**：1个月了解

### 数据中心网络（DCN）

**学习程度**：深入理解架构

- **传统三层架构**：核心层、汇聚层、接入层
- **Spine-Leaf架构**（现代数据中心标配）：
    - BGP EVPN + VXLAN技术
    - 无阻塞架构设计
    - 横向扩展能力
- **Overlay网络**：VXLAN隧道技术
- **学习时间**：2-3个月

- BGP邻居无法建立

- 网络延迟、丢包问题
- VLAN间通信失败
- **学习时间**：通过大量实验和实战积累（持续过程）

## **学习路线时间规划（全职学习）**

### 阶段一：基础筑牢（3-4个月）

1. HCIA/CCNA认证准备（2个月）
2. TCP/IP理论深入学习（1个月）
3. Linux基础命令（1个月）
4. Python基础语法（1个月）

### 阶段二：核心技能（4-5个月）

1. HCIP/CCNP认证准备（3个月）
2. BGP深入学习（1个月）
3. 防火墙技术（1个月）
4. Python网络自动化（2个月）

### 阶段三：专项提升（2-3个月）

1. 数据中心网络架构（1个月）
2. 华为云产品学习（1个月）
3. 实战项目练习（1个月）

如果是在职学习，时间需要延长到18-24个月。
